# Sydney House Price Analysis

## Overview
This project analyzes property transaction trends in Sydney using a Kaggle dataset. The goal was to identify **seasonal patterns in property sales** and understand when high-value and low-value property transactions occur throughout the year.

---

## Key Findings
- **Busiest Months for Transactions:**  
  November, May, and March, with **March being the peak month** for property transactions.

- **Least Active Month:**  
  January records the **lowest number of property transactions**.

- **Months for Expensive Property Sales:**  
  March, September, and November are the months when **high-value properties** are most frequently sold.

- **Months for Lower-Value Property Sales:**  
  January, July, and December see **more transactions of less expensive properties**.

These insights can help buyers, sellers, and investors better understand seasonal trends in the Sydney real estate market.

---

## Dataset
- Source: [Kaggle Sydney House Prices Dataset](https://www.kaggle.com/)  
- Contains property transaction data including:
  - Transaction date
  - Property price
  - Location
  - Property type

---

## Methodology
1. **Data Cleaning & Preprocessing:**  
   - Handled missing values  
   - Converted transaction dates to extract months  
   - Standardized property price and location data

2. **Exploratory Data Analysis (EDA):**  
   - Analyzed transaction counts per month  
   - Visualized average property prices across months  
   - Identified trends in high- and low-value property sales

3. **Insights Extraction:**  
   - Highlighted peak and off-peak months  
   - Identified months associated with high-value transactions
